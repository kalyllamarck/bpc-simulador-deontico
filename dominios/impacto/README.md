# Domínio: Impacto (Tela 2)

Jurista mexe na norma deôntica e o sistema estima o impacto no orçamento do BPC + acende um
**semáforo** sobre o equilíbrio do art. 201 (expõe a tensão, **não** declara inconstitucional).

Método KISS: cada parâmetro é uma "torneira".
**Impacto R$ = Δ beneficiários × 1 salário mínimo × 12.**
Parâmetros que pesam: limiar de renda (¼ → ½ SM), idade do idoso, conceito de deficiência.

Âncora real (2024): **R$ 102,1 bi** pagos a **5,8 milhões** de pessoas; benefício = 1 SM cheio.
